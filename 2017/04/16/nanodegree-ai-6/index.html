Propositional Logic

Given B E A M J standing for some symbol, so that

  • E or B implies A: (E v B) -> A
  • A implies J and M: A -> (J ^ M)
  • J if and only if M: J <–>M

Truth Table

Terminology

  • A statement is valid if it is true in every possible model
  • A statement is satisfiable if it is true in some models

Propositional Logic Limitations

  1. Can only handle true/false value, cannot handle Uncertainty
  2. Cannot talk about Objects which have properties, and cannot talk about relationships between objects
  3. No shortcut to succinctly talk about complex world

First Order Logic

An extension of propositional logic; first order means its on object, not on relations

Comparison

Syntax

  • Operators: ^, v, not, ->, <–> ()
  • Quantifiers: for all, there is one
